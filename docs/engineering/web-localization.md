# 網頁多語系機制 (Web Localization)

- **Radio Button Hack**：
  - 使用隱藏的 `input[type="radio"]` 作為語系切換器。
  - **結構要求**：Radio Button 必須與主要內容同層，且必須被具備 `role` 的 Landmark 元素（如 `<nav>`, `<header>`, `<main>`）包覆。
  - 使用 CSS 兄弟選擇器 (`~`) 配合 `:checked` 切換不同語系標籤的顯隱。
- **語意化與 Lang 屬性**：
  - 每一段文字內容必須同時提供 ZH (正體中文)、EN (英文)、JA (日文)、SC (簡體中文)。
  - 語言容器必須標註正確的 `lang` 屬性。
  - **A+ 規範**：語系切換標籤的 `lang` 屬性不可與 `<html>` 主語系相同。
- **在地化術語對齊**：
  - 臺灣：最佳化、快速鍵、應用程式、正體中文。
  - 大陸：优化、快捷键、应用程序、简体中文。
  - 日本：アプリケーション、携帯型 PC、タッチ キーボード。
