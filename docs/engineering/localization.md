# 在地化與術語規範 (Localization)

- **UI 空間適應**：
  - 日文與西歐語系較長，動態版面配置必須確保 `AutoSize` 與 `MaximumSize` 同時開啟以觸發自動換行。
- **變數與預留位置**：
  - 嚴禁翻譯 `{0}` 等預留位置。
  - 必須確保語法順序正確 (例如：`第 {0} 頁` vs `{0} ページ目`)。
- **助記鍵 (Mnemonics)**：
  - 跨語系按鈕字母應儘可能一致 (例如：確認為 `(A)`)。
  - **助記鍵冪等性**：生成提示前須執行冪等性檢查，避免重複後綴。
- **資源註釋與語境**：
  - `.resx` 檔案中必須填寫 `<comment>` 標籤，且註釋內容必須使用與該資源**相同的語系**。
- **設計工具保護 (Designer)**：
  - 允許在 Designer 中硬式編碼正體中文以供預覽。
  - 執行階段必須透過 `ApplyLocalization` 再次賦值以確保多語系正確性。
  - 嚴禁手動修改 `MainForm.Designer.cs` 中自動生成的版面配置結構。
