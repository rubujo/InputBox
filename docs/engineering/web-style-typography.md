# 網頁排版與標籤規範 (Web Style & Typography)

- **CJK 排版標準**：
  - **符號全形化**：正體中文、日文、簡體中文區塊必須使用全形 `／`、`（`、`）`。
  - **零空格原則**：全形符號與前後文字、數字、按鍵標籤之間**嚴禁留空格**。
- **按鍵標籤 (Keyboard Formatting)**：
  - 所有按鍵名稱、符號、數字必須包裹於 `<kbd>` 標籤內。
  - **格式規範**：
    - CJK：以首頁實際呈現格式為準，使用未加方括號的按鍵名稱（如 `<kbd>Alt</kbd>`、`<kbd>Enter</kbd>`），並搭配全形符號 `／`、`（`、`）`。
    - EN：採用標準 `Key` 格式（如 `<kbd>Alt</kbd>`），並使用半形標點符號。
  - **組合鍵**：加號 `+` 前後必須保留一個半形空格（如 `<kbd>Alt</kbd> + <kbd>F4</kbd>`）。
- **第三方披露與授權**：
  - 頁尾（Footer）必須披露第三方函式庫清單。
  - 清單必須與 `main` 分支之 `README.md`（權威來源）保持 100% 同步。
