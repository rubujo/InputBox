# 網頁排版與標籤規範 (Web Typography)

- **CJK 排版**：
  - 符號全形化：ZH, JA, SC 必須使用全形 `／`、`（`、`）`。
  - 符號與文字、數字、按鍵標籤之間**禁止留空格**。
- **按鍵標籤 (Keyboard Formatting)**：
  - 所有按鍵名稱、符號、數字必須包裹於 `<kbd>` 標籤。
  - **格式**：
    - CJK：`[按鍵]` (如 `<kbd>[Alt]</kbd>`)。
    - EN：`Key` (如 `<kbd>Alt</kbd>`)。
  - **組合鍵**：加號 `+` 前後必須保留一個半形空格 (如 `[Alt] + [F4]`)。
- **第三方披露 (Notices)**：
  - 頁尾必須包含第三方函式庫清單 (名稱、連結、授權)。
  - 清單必須與 `main` 分支之 `README.md` 保持 100% 同步。
